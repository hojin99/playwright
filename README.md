# playwright

## 환경

* e2e 테스트를 위한 프로젝트 생성  
`npm init playwright`  

* headless browser 사용을 위해 playwright 설치
`npm install playwright`  

## 실행

* static 서버 실행  
`node server`   

* playwright를 이용해서 static 서버로 부터 페이지를 로드하고 arjs 보고서 export
`node index.js`  

* playwright를 이용해서 e2e 테스트  
`npx playwright test`
